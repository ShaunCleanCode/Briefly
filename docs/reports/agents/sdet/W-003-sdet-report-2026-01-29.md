## 5) Root Cause Analysis

### Primary Issue: Route Interception Pattern
- **Problem**: Playwright `page.route('**/api/onboarding/**', ...)` 패턴이 실제 URL과 매치되지 않음
- **Symptom**: 모든 E2E 테스트에서 `waitForQuestion` timeout
- **Fix**: Glob 패턴을 regex로 변경: `/\/api\/onboarding\/.*/`

### Secondary Issues
| Issue | Root Cause | Fix |
|-------|-----------|-----|
| Dev server stale build | Next.js 번들 404 | 서버 재시작 |
| Back navigation timing | Client-side answerStack state | 명시적 wait 추가 |
| Multi-choice max limit | 4th option disabled | `isDisabled()` 체크 추가 |
